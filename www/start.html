Getting Started

Download and install OrgMassSpecR from either CRAN (the stable version) or R-Forge (the development version). RGui for Windows and R.app GUI for Mac OS X have menu systems for installing and updating packages, or use the install.packages function.

Load OrgMassSpecR in your current R session with the following command.

library(OrgMassSpecR)

Documentation and examples for each function are available. For example

?FragmentPeptide

will show the documentation for the function FragmentPeptide.

To quickly show the arguments for a function, use args.

args(MonoisotopicMass)

Information about the package and a list of included functions and data sets can be displayed with the following command.

help(package = "OrgMassSpecR")

Use the command vignette(package = "OrgMassSpecR") to view the package vignette.
