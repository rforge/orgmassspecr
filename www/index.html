OrgMassSpecR is an extension for the R statistical computing language. It contains functions to assist with organic/biological mass spectrometry data analysis.

This project is under continuous development. The stable version is on CRAN. The development version is on R-Forge, under R Packages.

Examples are in the package vignette (PDF).
