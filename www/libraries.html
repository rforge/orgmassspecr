Getting Started

Mass spectral libraries are provided as companion packages to OrgMassSpecR. The library content is currently focused on electron impact (EI) spectra of anthropogenic environmental contaminants and halogenated natural products. The libraries are intended to document and organize the evidence for compound identifications, and thus have a somewhat different focus than standard reference databases such as the NIST/EPA/NIH library. For example, the OrgMassSpecR libraries store fragment ion identifications obtained by the manual interpretation of mass spectra and allow organization based on compound class and other parameters. The libraries may also contain interesting but unidentified mass spectra.

Each library package generates a PDF report containing the annotated spectra, and a single text file containing all spectra in NIST MSP format. The PDF report allows the spectra and identification evidence to be viewed without specialized software. The MSP file can be used to import the spectra into other software, for example to create a custom searchable library in the NIST MS Search program. The PDF reports and MSP files are available below (through links to Dropbox). MD5 checksums are given to verify the file integrity. Within the R packages, the spectra and metatdata are stored in CSV formats for working with the data in R, and the packages may contain functions to generate summary tables and plots.

The packages are available through the OrgMassSpecR R-Forge webpage, under R packages.

California Sea Lion Contaminant Library (Example)

This library is provided as proof-of-concept. It contains a vignette (PDF) describing the general file structure and functionality of spectral library packages.

Package Name: SpecLibExample
Authors: Eunha Hoh and Nathan Dodder
Sample: Single sea lion blubber sample from a 4+ year old male stranded in southern California.
Instrument: GCxGC-TOF in EI mode.
Compounds: 35 anthropogenic contaminants, 3 suspected natural halogenated products, and 1 unknown.
PDF Report: SpecLibExample.pdf. 595 KB. MD5 checksum: aa07e28848b7bff311d6dcdeeb097fc1
NIST MSP Text Format: SpecLibExample.txt. 49 KB. MD5 checksum: 31da4996d34b7c6da9635608261b5890
